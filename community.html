    BuscasTorneos
    ¿Cómo subo mis torneos?

    Tienes que darme los datos de tu torneo en mi correo, tik tok, o discord, pero antes, para poder mandar un torneo, necesitas tener 12 tukins, que se consigue mediante omlet arcade refer, pero, puede ser de cualquier juego, aunque te recomiendo que hagas de minecraft.

  • Para torneosnesesitas datos
    ¿Cuáles datos?

    Los datos son los siguientes:
    1- Imagen / poster etc de tu torneo
    2- Fecha y hora (la hora solamente de tu país, yo la saco para la mayoría de países de Latinoamérica)
    3- Pequeña descripción
    4- Link (si es un servidor de minecraft o de algún juego que necesites prenderlo, pero ya puedes tener el link antes.)
    5- Versión del juego (no del torneo, sino del juego, por si algunos tienen una versión anterior o posterior a la tuya)
    6- Tu nombre de usuario (en aplicación o página que tú quieres que te llamen, preferible no usar número telefónico, usar tik tok o discord)
    Con estos datos aceptaré tu torneo y la pondré aquí.
